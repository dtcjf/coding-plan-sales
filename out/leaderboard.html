评测方法说明

了解我们如何评估和比较不同 AI 模型的编程能力

LM Arena 评测

  • • 基于真实用户的 A/B 对比测试
  • • 评测代码质量、可读性和实用性
  • • 涵盖多种编程语言和任务类型
  • • 样本量大，统计结果可信度高

SWE-bench Verified

  • • 基于真实 GitHub Issue 的测试集
  • • 验证模型修复实际 bug 的能力
  • • 包含代码理解、定位和修改能力
  • • 通过单元测试验证解决方案正确性
