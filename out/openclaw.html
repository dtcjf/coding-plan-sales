接入 OpenClaw 教程

OpenClaw 是一个开源的 AI 编程助手，支持连接多种大模型 API，包含 MiniMax、Moonshot、智谱、阿里云百炼、火山引擎等国内提供商。

安装 OpenClaw

快速开始（推荐）

macOS / Linux
curl -fsSL https://openclaw.ai/install.sh | bash
Windows (PowerShell)
iwr -useb https://openclaw.ai/install.ps1 | iex

其他安装方式

npm 全局安装: npm i -g openclaw
Git 克隆安装: git clone https://github.com/openclaw/openclaw.git && cd openclaw && pnpm install && pnpm run build

初始化配置

openclaw onboard

运行 openclaw onboard 启动配置向导，按提示完成 API Key 配置。

手动配置文件方式

除了使用 openclaw onboard 向导，你也可以直接编辑配置文件：

配置文件位置: ~/.openclaw/openclaw.json
{
  "models": {
    "providers": {
      "custom-provider": {
        "baseUrl": "https://api.example.com/v1",
        "apiKey": "${API_KEY}",
        "api": "openai-completions",
        "models": [
          {
            "id": "model-id",
            "name": "Model Name",
            "contextWindow": 128000,
            "maxTokens": 32000
          }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": { "primary": "custom-provider/model-id" }
    }
  }
}
